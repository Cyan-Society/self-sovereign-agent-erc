# Letta

> Letta is an open source framework for building stateful AI agents with memory and reasoning capabilities.

Letta enables developers to create AI agents that can remember context across conversations, manage their own memory, and perform complex multi-step tasks. The framework provides tools for memory management, tool execution, and multi-agent coordination.
